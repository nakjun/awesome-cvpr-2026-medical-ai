# Contributing

This repository is generated from CVPR 2026 virtual conference metadata, then lightly curated through keyword rules for medical AI topics.

## Update the Catalog

Run the generator from the repository root:

```powershell
python build_cvpr2026_medical_ai_catalog.py --enrich-details --enrich-external-links
```

This refreshes:

- `README.md`
- `medical_ai_catalog/cvpr2026_medical_ai_papers.json`
- `medical_ai_catalog/cvpr2026_medical_ai_papers.csv`
- `medical_ai_catalog/modality_index.md`
- `medical_ai_catalog/resource_index.md`
- `medical_ai_catalog/stats.md`

## Curation Notes

The catalog is keyword-assisted, not a final peer-reviewed taxonomy. When editing the filter rules, check `match_patterns` in the JSON file for borderline entries.

Good additions include:

- missing medical imaging papers
- corrected modality labels
- missing GitHub, project, demo, video, or arXiv links
- false-positive removals with a short reason

Please keep the README generated by the script so future updates remain reproducible.
